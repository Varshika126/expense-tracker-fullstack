# 💰 Expense Tracker - Full Stack Web Application

## 🚀 Tech Stack
- Spring Boot
- Spring Security
- Thymeleaf
- MySQL
- Hibernate / JPA
- Chart.js
- HTML, CSS

## 🔐 Features
- User Registration & Login
- Role-Based Authentication
- Add / Edit / Delete Expenses
- Monthly Expense Filter
- Category-wise Pie Chart
- Monthly Analytics Bar Chart
- Multi-user data isolation

## 📊 Project Description
A full-stack expense management system that allows users to track their daily expenses with authentication and dynamic analytics.
